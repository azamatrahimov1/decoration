После распаковки веб сайт запускается через файл index(html).
Для создания данного сайта использовались html и css, для респонсивности в мобильной версии использовался JavaScript.
Если изображения не прогружаются или внешность сайта не соответствует данным скриншотам , требуется в коде каждого из файлов указать путь расположения
